# これは何?

サークルのWebAPIを使ったゲーム製作講座用に書いている資料です。
対象読者は何らかのプログラミング言語を少し触ったことはあるけど、
Pythonははじめてという方を対象にしています(ちなみにサークルでは最初にC言語をします)

# 読むとどうなるの?

WebAPIとゲームを連携させて簡単なオンラインゲームが作れるようになります。
